# Politica Globale

La politica postbellica è frammentata, locale e spesso instabile. Le grandi strutture statali prebelliche non esistono più; al loro posto operano entità regionali, fazioni e poteri informali.

Questa sezione descrive l’evoluzione delle strutture politiche europee, mediterranee e globali.

Panoramica delle strutture politiche, governi, alleanze e conflitti mondiali. Questa sezione fornisce contesto per le fazioni, i conflitti locali e le interazioni socio-politiche.
